## 📦 Estruturas Base e Laboratórios

Para facilitar a navegação, os workflows de estudo foram divididos por conceitos arquiteturais:

### 🧠 Integrações com IA e Agentes
* `AIAgent.json` / `LLMAutomacao.json`: Estruturas base de agentes conversacionais e automações de rotina com LLMs.
* `AnaliseSentimento.json`: Processamento de linguagem natural para classificação de textos.

### 🔄 Pipelines de Dados e ETL
* `ETL.json`: Fluxo de extração, transformação e carregamento de dados estruturados.
* `HTMLReport.json`: Geração automatizada de relatórios visuais.
* `APIRest.json` / `ConsultaBasica.json`: Consumo, paginação e manipulação de APIs externas.

### 🔀 Roteamento e Manipulação de Dados (Data Wrangling)
* `MergeFork.json` / `MergebyAppend.json` / `Mergebyindex.json` / `Mergebykey.json`: Técnicas avançadas de junção de múltiplos arrays e consolidação de pacotes JSON.
* `Split.json`: Separação de grandes volumes de dados em lotes (Batching).
* `If_switch.json`: Roteamento condicional e lógicas de desvio.

### ⚡ Triggers, Eventos e Resiliência
* `Webhook.json` / `Form.json`: Portas de entrada para recebimento de dados externos em tempo real.
* `Cron.json` / `Interval.json`: Agendamento de rotinas e tarefas em background.
* `Fallback.json`: Estrutura de tratamento de erros e exceções, garantindo a resiliência das automações.
* `HumanLoop.json`: Intervenção e aprovação humana no meio do processo automatizado (Human-in-the-loop).

### 🔐 Segurança e Atendimento
* `BasicAuhtentic.json`: Laboratório de métodos de autenticação para consumo de APIs protegidas.
* `Chamados.json` / `ChamadosUrgentes.json`: Estrutura de roteamento e triagem de tickets de suporte.
